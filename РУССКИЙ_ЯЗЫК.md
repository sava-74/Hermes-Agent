## 🌐 Как работает перевод

### Автоматически

Скрипт `herLayStart.bat` устанавливает:
```batch
set HERMES_LANGUAGE=ru
```

### В коде Python

```python
from hermes_cli.translations import t, is_russian

# Простой перевод
text = t("BANNER.welcome", "Welcome")
# Результат: "Добро пожаловать в Hermes Agent"

# С форматированием
text = t("BANNER.update_available", "Update available", count=5)
# Результат: "⚠ 5 коммитов позади"

# Проверка языка
if is_russian():
    print("Привет!")
else:
    print("Hello!")
```

Откройте `herLayStart.bat` и измените:

```batch
REM Было:
set HERMES_LANGUAGE=ru

set HERMES_LANGUAGE=en
```

## ❓ Решение проблем

### Ошибка: "Виртуальное окружение не найдено"

**Решение:**
```cmd
cd d:\GitHub\agent-1
uv venv .venv --python 3.12
.venv\Scripts\activate
uv pip install -e ".[all]"
```

### Ошибка: "Python не найден"

**Проверка:**
```cmd
dir .venv\Scripts\python.exe
```

Если файла нет, создайте venv заново (см. выше).

### Переводы не применяются

**Проверка:**
1. Убедитесь, что `HERMES_LANGUAGE=ru` установлен
2. Перезапустите Hermes
3. Проверьте файл `hermes_cli/translations/ru.py`

### Баннер на английском

**Причина:** Скрипт не загружает переводы

**Решение:** Используйте `herLayStart.bat` вместо прямого запуска `hermes`
