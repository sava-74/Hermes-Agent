## Технические решения

### API-сервер (Этап 1)

```python
# Минимальный пример (изучено в run_agent.py):
from run_agent import AIAgent

agent = AIAgent(
    provider="alibaba",
    model="qwen3.5-plus",
    api_key="твой_ключ",  # или из .env
    enabled_toolsets=["hermes-cli"],  # все инструменты
)

response = agent.chat("Привет!")
```

| Решение | Выбор | Обоснование |
|---------|-------|-------------|
| Фреймворк | FastAPI | Современный, асинхронный, автодокументация |
| Протокол | HTTP + WebSocket | REST для запросов, WS для стриминга |
| Запуск | Служба Windows (NSSM) | Автозапуск, работа в фоне |

### Chat UI (Этап 2)

| Решение | Выбор | Обоснование |
|---------|-------|-------------|
| Технология | Python + CustomTkinter | Быстро, кроссплатформенно, в экосистеме |
| Дизайн | Современный, минимализм | Как ChatGPT / Claude |
| Подключение | WebSocket к API | Стриминг в реальном времени |
